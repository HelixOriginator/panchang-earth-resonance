PERI
Panchang-Earth Resonance Intelligence

A global open research framework exploring whether multidimensional celestial geometry encoded in Panchang systems contains statistically measurable relationships with Earth's geophysical, biological, climatic, and civilizational systems.

Overview

PERI (Panchang-Earth Resonance Intelligence) is an interdisciplinary research initiative that combines:

Ancient Panchang astronomical geometry
Modern geophysics
Earth system science
Historical datasets
Machine learning
Statistical inference

The project treats Panchang as a high-resolution celestial coordinate language rather than an astrological system.

The central objective is to investigate whether specific multidimensional Earth-Moon-Sun configurations leave measurable statistical signatures across:

Seismic activity
Climate systems
Ionospheric dynamics
Agricultural cycles
Historical and civilizational transitions
Core Hypothesis

Certain recurring multidimensional configurations of the Earth-Moon-Sun system create measurable resonance states with terrestrial systems, detectable as statistical signatures in geophysical, biological, and historical datasets when tested rigorously as systems science.

PERI does not assume causation.

It focuses exclusively on:

Correlation testing
Statistical validation
Predictive utility
Reproducible scientific methodology
